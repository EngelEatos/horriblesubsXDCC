# horriblesubsXDCC

## Installation

```python -m pip install -r requirements.txt```

## Usage

1. set animes you watch in ```settings/animes.json``` or ```python sub_gui.py```
2. edit irc config ```settings/irc.sjon``` (user, res, etc.)
3. start downloading ```python horriblesubs.py```
